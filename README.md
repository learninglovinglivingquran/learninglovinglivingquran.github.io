# learninglovinglivingquran.github.io

# App
The app is meant to slightly guide the natural interaction with the Quran.
The LQ app works in stages:

### 1. Select the Quran Paths and Order
Currently, a reader of the Quran has basically three options when they're starting to the read the Quran. 
- They can start at the beginning, and start reading from the beginning of the book order of the Quran
- They can select a random surah, and start reading from that surah
- They can go to a random page and ayah of the Quran, and start from there
With the LQ app, we will try to enhance the options above by offering readers additional paths, which will consist of ayaat grouped together for specific topics.
With each path, the reader can also select if they would like to learn the ayaat in the default book order, ayaat for the topic from a specific surah, or learn the related ayaat in a random order.

**Possible Paths**
- Letters from my Master
  - default order
  - Select a Surah
  - Random selection
- Consolation from my Master
  - default order
  - Select a Surah
  - Random selection
- Counsel from my Master
  - default order
  - Select a Surah
  - Random selection
- Mentorship from my Master
  - default order
  - Select a Surah
  - Random selection
- The Beloved of my Master
  - default order
  - Select a Surah
  - Random selection
The Language for my Master    

### 2. After selecting a path and order, select a block
Once a path is selected with an order, the reader will be presented with a selection of blocks, which are selections of ayaat and commentary.
In the default and surah order, the reader will be presented with blocks in the default order.
In the random selection order, the reader will be presented with blocks randomly organized into a custom order. Each time the reader revisits the path in the random selection order, the blocks with organized in a new random order.

### 3. After selecting a block, read the ayaat and commentary
When a block is selected, the reader will be presented with one or more ayaat, with translation and commentary

### 4. After completing the block reading, answer the quiz questions
Along with a section of reading in the block, there were be accompanying questions. In order to complete the block, the reading must be completed along with the questions in the block.

### 5. Repeat daily
To achieve the goal of living with the Quran, this process should be repeated on at least a daily basis to create a consistent connection with the Quran, which should hopefully result in manifestation of the Quran in the reader's life.
